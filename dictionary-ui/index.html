    Как это работает

    Если в PDF есть текстовый слой, индекс собирается автоматически и поиск работает как по малайским заголовкам, так и по русским переводам.

    Если PDF состоит только из сканов без текста, интерфейс останется рабочим в режиме просмотра страниц. Для полного поиска понадобится OCR в отдельном шаге.
